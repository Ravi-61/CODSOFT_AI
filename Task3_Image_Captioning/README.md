# Image Captioning AI

This project generates captions for images using a pre-trained BLIP (Bootstrapping Language-Image Pretraining) model.

## Features

- Automatic image caption generation
- Uses Hugging Face Transformers
- Uses Salesforce BLIP model
- Combines Computer Vision and NLP

## Technologies Used

- Python
- Transformers
- PyTorch
- Pillow

## Installation

pip install transformers
pip install torch
pip install pillow

## Run

python image_captioning.py

## Example Output

Input Image:
Dog playing in a park

Generated Caption:
a dog running through a grassy field
